# kotlin-attendance-precourse

1. 파일에서 출석 기록 가져오기
2. 가이드 출력
3. 출석 확인
    1. 입력받기(검증)
    2. 출석기록에 추가
    3. 출력
4. 출석 수정
    1. 입력 받기(검증)
    2. 출석 기록에서 찾기(검증)
    3. 출석 기록에서 타겟 수정
5. 크루별 출석 기록 확인
    1. 입력받기(검증)
    2. 출석 기록 출력
    3. 출석, 결석, 지각 판정
    4. 경고 판정
6. 제적 위험자 확인
    1. 출석 기록에서 대상 찾기
    2. 지각 횟수를 결석으로 치환
    3. 대상자 정렬
