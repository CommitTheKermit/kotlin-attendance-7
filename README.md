# kotlin-attendance-precourse

1. 파일에서 출석 기록 가져오기
2. 출석 확인
    1. 안내문 출력
    2. 입력받기(검증)
    3. 출석기록에 추가
    4. 출력
3. 출석 수정
    1. 안내문 출력
    2. 입력 받기(검증)
    3. 출석 기록에서 찾기(검증)
    4. 출석 기록에서 타겟 수정
4. 크루별 출석 기록 확인
    1. 안내문 출력
    2. 입력받기(검증)
    3. 출석 기록 출력
    4. 출석, 결석, 지각 판정
    5. 경고 판정
5. 제적 위험자 확인
    1. 안내문 출력
    2. 출석 기록에서 대상 찾기
    3. 지각 횟수를 결석으로 치환
    4. 대상자 정렬
