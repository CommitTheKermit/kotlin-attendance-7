# kotlin-attendance-precourse
# 기능 목록 

1. 출석표 파일 읽기
2. 실행 가이드 출력
3. 출석 관련 처리
4. 출석 상태 판정
5. 출석 확인
    1. 가이드 출력
    2. 입력 받기
    3. 출석표 추가
6. 출석 수정
    1. 가이드 출력
    2. 입력 받기
    3. 출석표 수정
7. 크루별 출석 기록 확인
    1. 가이드 출력
    2. 입력 받기
    3. 출석 기록 출력
    4. 경고, 면담, 제적 판정
8. 제적 위험자 확인
    1. 제적 대상자, 면담 대상자, 경고 대상자순으로 출력
9. 종료
